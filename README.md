# 广州地铁活点地图 🧹

⚡ **广州地铁活点地图** - 基于哈利波特活点地图风格的广州地铁站点记录应用

## 功能特点

### 🧹 活点地图主题
- 输入咒语 **「我庄严宣誓我不干好事」** 打开地图
- 羊皮卷缓缓展开的动画效果
- 魔法风格的界面设计

### ✨ 点亮站点
- 点击站点记录你的到访
- 已点亮的站点显示飞天扫把 🧹
- 到访5次以上的站点升级为 **金色飞贼** 🌟

### 📝 记录回忆
- 记录到访时间
- 记录同行的人
- 记录做了什么事情

### 📍 站点名称显示
- 迷雾站点显示站点名称
- 点亮后名称自动隐藏
- 智能防重叠布局

### 🎮 交互功能
- 滚动条浏览地图
- 鼠标滚轮缩放
- 折叠式控制面板

## 使用说明

1. 打开网页，输入咒语：**我庄严宣誓我不干好事**
2. 点击或滑动地图探索广州地铁
3. 点击站点添加到访记录
4. 尝试让一个站点达到5次到访，解锁金色飞贼！
